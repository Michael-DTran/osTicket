# Purpose of this Lab
- To gain more experience with Azure
- Experience as both the admin and the user of the ticket system
- Gain a better understanding of all the parts of the ticketing system
  * Ticket properties
  *  SLA (Service-Level Agreement)
  *   Departments
  *   Permissions
  *   Users
